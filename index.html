⚙️ 선생님 설정

학생들의 답안을 구글 시트로 자동 수집할 수 있어요.

📌 "보내기 → 링크" 에서 복사한 URL을 붙여넣기

🔑 폼 필드 entry ID (각 칸의 입력 위치)

각 입력 칸의 entry.숫자 ID를 입력하세요. 아래 "자동 찾기" 버튼을 사용하면 편해요.

🧹 학생 데이터 초기화

이 브라우저에 저장된 모든 학생의 학습 기록을 한 번에 삭제해요.
⚠️ 학생들에게 파일을 보내기 전 또는 새 학기 시작 시 사용하세요.

✓ 삭제됨: 학생 이름, 학습 점수, 답안 기록
✓ 유지됨: 구글 폼 설정 (선생님 설정은 그대로)

📝 1단계: 구글 폼 만들기

  1. forms.google.com에서 새 폼 만들기
  2. 다음 7개 질문 추가:
    1. 이름 (단답형)
    2. 날짜 (단답형)
    3. 주제 (단답형)
    4. 점수 (단답형)
    5. 진행도 (단답형)
    6. 내가 만든 사전 (장문형)
    7. 학습 답안 (장문형)

🔑 2단계: entry ID 자동 찾기 (가장 쉬운 방법!)

  1. 폼 편집 화면 오른쪽 위 점 3개 메뉴 (⋮) 클릭
  2. "양식 미리 작성" 또는 "답변 미리 채우기" 선택 (메뉴 이름이 다를 수 있어요)
  3. 각 칸에 다음 표시값을 정확히 입력:
    • 이름 → __이름__
    • 날짜 → __날짜__
    • 주제 → __주제__
    • 점수 → __점수__
    • 진행도 → __진행도__
    • 사전 → __사전__
    • 답안 → __답안__
  4. 아래 "링크 가져오기" → "링크 복사"
  5. 설정 탭의 "🔍 자동 찾기 도우미" 버튼 누르고, 복사한 링크 붙여넣기
  6. 나머지는 자동으로 채워짐! ✨

📊 3단계: 구글 시트 연결

  1. 구글 폼 → "응답" 탭 클릭
  2. 우측 위 스프레드시트 아이콘 클릭 → 새 시트 생성
  3. 이제 학생 답안이 시트에 자동 누적돼요!

📋 4단계: 폼 URL 복사

  1. 폼 편집 화면 오른쪽 위 "보내기" 클릭
  2. 🔗 링크 아이콘 클릭
  3. 전체 URL 복사 (단축 URL 체크박스는 해제)
  4. 설정 탭의 "구글 폼 URL" 칸에 붙여넣기

✨ 학생 학습 후

학생이 학습 끝나고 "선생님께 제출" 누르면 자동으로 구글 시트에 한 줄 추가돼요. 학생은 폼 화면을 보지 않고 백그라운드에서 전송됩니다.
